Finally, before looking at January‚Äôs tweets, I‚Äôm deriving the tweet_type variable so that information about retweets, replies, and original tweets is all in one variable. To keep it simple, if a tweet has data in the reply_to_screen_name field and it‚Äôs not tto myself, I‚Äôm counting it as a reply. I like to thread too much to assume a tweet replying to myself is actually part of a conversation with n > 0 other people üò¨.

library(forcats)

my_tweets <- my_tweets %>%
  mutate(tweet_type = case_when(is_retweet ~ "Retweet",
                                !is.na(reply_to_screen_name) & reply_to_screen_name != "sharlagelfand" ~ "Reply",
                                TRUE ~ "Original"),
         tweet_type = fct_relevel(as_factor(tweet_type),
                                  c("Original", "Reply", "Retweet"))) %>%
  select(-is_retweet)

Every single one of these people likes and uses R. But would it really be a Twitter analysis if I didn‚Äôt do some text analysis? I‚Äôm going to do tf-idf (using Julia Silge and drob‚Äôs tidytext package), 1) because I have blogged about it before and can copy my old code, but mostly 2) because it‚Äôs a fun way to look at differences in how people communicate.

I‚Äôm throwing myself back in the mix (hence keeping getting the top 8 before) to see the words that each one of us uses the most, but everyone else uses‚Ä¶ not so much.

There‚Äôs a bunch of data cleaning here to replace quotes that aren‚Äôt quotes (i.e., ‚Äô ‚Äì Hadley was the most guilty of this), remove mentions of other people, and üò≠ remove emojis. I know, I know. Another time.

library(tidytext)

tidy_words <- friends_tweets %>%
  select(username, text) %>%
  bind_rows(
    january_tweets %>%
      select(text) %>%
      mutate(username = "sharlagelfand")
  ) %>%
  mutate(text = str_replace_all(text, "‚Äô", "'")) %>% 
  unnest_tokens(word, text, token = "tweets") %>%
  filter(!str_detect(word, "^@"),
         !str_detect(word, "[\\uD83C-\\uDBFF\\uDC00-\\uDFFF]+")) %>%
  anti_join(stop_words, by = "word") %>%
  count(username, word)

total_words <- tidy_words %>%
  group_by(username) %>%
  summarize(total = sum(n))

tf_idf_words <- tidy_words %>%
  left_join(total_words, by = "username") %>%
  bind_tf_idf(word, username, n) %>%
  group_by(username) %>%
  top_n(5, wt = tf_idf) %>%
  ungroup() 

tf_idf_words %>%
  ggplot(aes(x = fct_reorder(word, tf_idf), 
             y = tf_idf, 
             fill = username)) +
  geom_col() + 
  labs(x = NULL, y = "tf-idf") +
  facet_wrap(~username, ncol = 3, scales = "free") + 
  coord_flip() + 
  theme(axis.text.x = element_blank(),
        legend.position = "none",
        axis.text = element_text(size = 10),
        strip.text = element_text(size = 10))
